# Testing Slash Commands

This is a test PR to demonstrate the new slash commands:

## New Commands Available

- `/generate_description` - Auto-generates and updates the PR description
- `/generate_reports` - Generates a combined report with summary, review, and suggestions in one comment

## Test Instructions

1. Comment `/generate_description` to test PR description generation
2. Comment `/generate_reports` to test combined report generation

## Expected Results

- The PR description should be automatically updated with AI-generated content
- A single comprehensive comment should be posted with all analysis combined
